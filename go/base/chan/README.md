| chan 操作         | nil   | 非空非满 chan    | 空的无缓冲 chan             | 空的有缓冲 chan             | 满 chan          |
| ----------------- | ----- | ---------------- | --------------------------- | --------------------------- | ---------------- |
| rx 从 chan 中接收 | 阻塞  | 正常接收, 非阻塞 | 阻塞                        | 阻塞                        | 正常接收, 非阻塞 |
| tx 向 chan 中发送 | 阻塞  | 正常发送, 非阻塞 | 阻塞                        | 正常发送, 非阻塞            | 阻塞             |
| 关闭 chan         | panic |                  | 读空 chan, 返回零值和 false | 读空 chan, 返回零值和 false |                  |

```go
var ch chan int
fmt.Println(ch) // nil
```

- 向一个关闭的通道发送值, 会导致 panic
- 从一个关闭的通道接收值, 会一直接收, 直到通道空
- 从一个关闭的空通道接收值, 会收到零值和 false (表示接收失败)
- 重复关闭通道, 也会导致 panic
